# Zen Barber - Next.js

Site migrate vers **Next.js App Router** avec medias locaux (photos/videos), logo en header/footer et favicon.

## Prerequis

- Node.js 20+
- npm

## Lancer en local

1. Installer les dependances

```bash
npm install
```

2. Lancer le serveur de dev

```bash
npm run dev
```

3. Build de production

```bash
npm run build
npm run start
```

## Structure

- `src/app` : routes Next.js App Router
- `src/components` : composants UI
- `public/images` : vraies images et videos du projet
